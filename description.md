# Общее описание
Основной стек технологий это react, redux и pixijs.

точка входа в приложение это файл `/src/lobby/index.js`. 
В этом файле подключается редьюсер для lobby и коннектиться главный компонент lobby `/src/lobby/App.js`

В файле `/src/lobby/App.js` через `react-router` подключаются компоненты с играми `/:sysname`

в `/src/slots` два основных каталога это 
- `/common` там храниться общие компоненты, редьюсеры и т.д. для всех игр
- `/games` общая папка и играми. Все файлы для игр включая js добавляются именно в эту папку.

Название для каталога с игрой дается в нижнем регистре без пробелов и других спецсимволов

Например название игры ``Spike`s Nite Out`` каталогу необходимо дать имя ``spikesniteout``
Что бы игра была доступна в игровом зале нужно добавить запись в редьюсер
``/src/lobby/reducers/lobby.js`` в свойство ``listAvailableGames`` добавить sysname игры

Новая игра создается на основе уже существующей, в данном случае на основе игры ``Alley Cats``.

Внутри игровой папки несколько каталогов
- ``/components`` собственные компоненты необходимые для игры
- ``/img`` каталог с изображениями
- ``/sounds`` звуки для игры

В корне папки с игрой есть файл ``config.js`` это конфиг файл для текущей игры.
В этом файле указываются все настройки для игры: позиционирование изображении,
настройки анимации символов и т.д. Все настройки записываются в объект ``settings``
который экспортируется в файл ``src/slots/games/*sysname*/components/View.js``

В конфиг файле можно выделить несколько ключевых свойств
- ``reel`` содержит настройки компонента с барабанами(их колличество, размеры и позицию)
- ``symbols`` содержит информацию о специальных символах. Например:
`freegameSymbol` символ для старта бонусной игры,
`freespinsSymbol` символ для старта бесплатных вращений)
- ``symbolAnimation`` - позиция компонента с анимацией символов и колличество кадров для каждой анимации
- ``animation`` содержит настройки для каждой анимации в игре. Пример:
```
symbol_1_anim: {   // файл анимации
  default: {       // имя анимации
    from: 0,       // номер кадра начинала анимации
    to: 60,        // номер кадра окончания анимации
    loop: true     // зацикливать анимацию или нет
  }
}
```
- ``controls`` настройки для кнопок управления 
- ``lines`` настройки для линий (общее колличество, толщина линий, позиция контейнера с линиями)
- ``freegame`` - настройки для компонента бонусной игры
- ``freespins`` - настройки для отображения бесплатных вращений
- ``infoWindows`` - настройки для информационных окон при старте бесплатных вращений и бонусной игры
- ``soundManifest`` - список звуков

##Главный экран 
Что бы создать главный экран игры. Нужно работать с config файлом,
дать имена для анимации символов по анологии с предыдущими играми.
Создать спрайт файлы (Удобно использовать AdobeAnimate он позволяет
экспортировать анимацию в спрайты.)
 
Этапы готовности главного экрана
- Все кнопки меню отпозиционированы в соответствии с макетом 
- В барабанах использвются нужные картинки символов 
- При клике по кнопке спин вращаются все барабаны
- При остановке барабана и выигрыше. Показывается анимация символов и выигрышные линии
